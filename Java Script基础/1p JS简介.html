JavaScript诞生于1995年，出现是主要用于处理网页中的前端验证

前端验证 指的时检查用户输入的内容是否符合一定的规则

比如用户名的长度 密码的长度 邮箱的格式

前身：leves script

可以用于前端 也可以用于服务器

ECMAScript是javaScript标准 所以一般情况下这两个词我们认为是一个意思

一个完整的javascript由一下三个部分构成

一丶ECMAScript 标准 二丶DOM 文档类型模型 让我们可以去操作网页 三丶BOM 浏览器对象模型 让我们可以操作浏览器

JS特点 解释型语言 类似于C和java的语法结构 动态语言 基于原型的面向对象
